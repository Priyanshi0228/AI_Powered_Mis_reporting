MIS Performance Analysis System

A simple Flask-based web application to manage company financial data and analyze performance using calculated MIS scores.

Features
Add company financial details (R&D, Admin, Marketing, Profit)
Automatic calculation of:
Revenue
Expenses
Profit %
MIS Score

Performance classification:
High Performer
Stable Growth
Under Observation

JSON API for data access
Delete company records

Tech Stack
Python (Flask)
HTML (Templates)
